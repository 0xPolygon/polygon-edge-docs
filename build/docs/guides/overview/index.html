Polygon SDK

Polygon SDK is a modular and extensible framework for building Ethereum-compatible blockchain networks.

This repository is the first implementation of Polygon SDK, written in Golang. Other implementations, written in other programming languages might be introduced in the future. If you would like to contribute to this or any future implementation, please reach out to the Polygon team.

To find out more about Polygon, visit the official website.

caution

This is a work in progress so architectural changes may happen in the future. The code has not been audited yet, so please contact Polygon team if you would like to use it in production.
